# 项目说明

## 安装依赖

1. 双击运行 `安装.bat` 脚本，该脚本会：
   - 设置npm镜像为淘宝镜像
   - 全局安装http-server

## 运行项目

1. 双击运行 `运行.bat` 脚本，该脚本会：
   - 切换到client_admin目录
   - 启动http-server服务器

2. 在浏览器中访问 `http://localhost:8080` 即可进入登录页面

## 功能说明

### 登录功能
- 用户名密码登录
- 滑动验证码验证

### 注册功能
- 账号、密码、昵称、邮箱、手机号等信息填写
- 身份选择下拉框
- 表单验证

## 技术栈

- HTML5
- CSS3
- JavaScript
- LayUI框架
- Axios
- http-server
